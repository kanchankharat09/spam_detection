This project implements a Spam Detection system using Machine Learning to classify messages as Spam or Not Spam (Ham). It uses text preprocessing and NLP techniques to train a classification model.


Project File
spam_detection_project_ml.ipynb

Approach

Loaded and explored the spam dataset
Cleaned and preprocessed text data
Converted text into numerical features using text vectorization
Trained a Machine Learning classification model
Evaluated model performance using accuracy and metrics


Technologies Used

Python
Google Colab
Pandas, NumPy
Scikit-learn


How to Run (Google Colab)

Open Google Colab
Upload the file:
spam_detection_project_ml.ipynb
Run all cells sequentially
The model will classify messages as Spam or Ham

Output

Predicts whether a given message is Spam or Not Spam
Displays model accuracy and evaluation results
